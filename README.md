# 🚀 Olayiwola Web Authentication & Dashboard

> *"Code with purpose. Design with faith. Build with Olayiwola."*

A beautiful, responsive, and accessible front-end web application featuring a complete user authentication flow and a dynamic interactive dashboard. This project demonstrates modern UI/UX principles, client-side validation, and state management using vanilla web technologies.

## 🔗 Live Demo
**[Click here to view the live project on GitHub Pages!](https://olajerry11.github.io/Olayiwola-signup/)** *(Note: If your repository is named something else, update this link!)*

## ✨ Key Features
* **Complete User Flow:** Seamlessly navigate between Signup, Login, Password Reset, and a secure Dashboard.
* **Interactive Dashboard:** A dynamic, tab-based workspace featuring tool panels (Projects, Templates, Analytics) and a daily devotion, all without reloading the page.
* **Smart Validation:** Custom JavaScript validation using Regular Expressions (Regex) to ensure proper email formatting and secure passwords.
* **Data Persistence:** Utilizes browser `localStorage` to save the user's name during signup and provide a personalized greeting on the dashboard.
* **Accessibility Focused:** Built with semantic HTML5 tags (`<main>`, `<header>`, `<nav>`), ARIA labels, and fully keyboard-accessible buttons (like the password visibility toggle).
* **Premium UI/UX:** Features a custom color palette (Deep Purple & Gold), smooth CSS transition animations, and simulated network loading states for a realistic application feel.
* **Fully Responsive:** Adapts flawlessly to desktop, tablet, and mobile screens.

## 🛠️ Technologies Used
* **HTML5:** For semantic, accessible structure.
* **CSS3:** For styling, flexbox layouts, custom gradients, and hover animations.
* **Vanilla JavaScript:** For DOM manipulation, form validation, and tab logic.

## 📁 Project Structure
* `index.html` - The Signup page (Entry point)
* `login.html` - The Login page with simulated authentication
* `dashboard.html` - The interactive user workspace and tools
* `forgot.html` - The Password Reset request page
* `style.css` - Global stylesheet for consistent branding
* `script.js` - Global JavaScript for form validation and logic
* `olayiwola-logo.png` - Brand asset

## 💻 How to Run Locally
1. Clone this repository: `git clone https://github.com/Olajerry11/Olayiwola-signup.git`
2. Open the project folder.
3. Double-click `index.html` to launch the application in your default web browser. No server setup required!
